Maternal Health Risk Workflow With Prefect

We built a machine learning solution to predict health risks, where K-Nearest Neighbors (KNN) achieved the best performance with 89% accuracy and an AUC of 0.93. Age was the most influential factor in predicting pregnancy-related health risks.

To ensure scalability, we deployed the KNN model using Docker and Flask, and automated the workflow with Prefect, creating a robust pipeline for seamless predictions and model updates. Key insights and actionable recommendations are provided for practical applications.

Wine Quality Classification

In this project, we classified wine quality using the UCI Wine Quality dataset, aiming to predict whether wines fall into lower or higher quality categories based on their chemical properties. We trained six models: K-Nearest Neighbors, Random Forest, Gradient Boosting, LightGBM, and XGBoost.

The Random Forest model emerged as the top performer, boasting an impressive 81% accuracy and a ROC-AUC score of 0.897. While it encountered a few false alarms, it excelled in precision and recall across all classes. To make this valuable model accessible for real-world applications, we deployed it locally using Flask, paving the way for effective wine quality classification.

Anomaly Detection in IDS

DThis project focuses on data from an intrusion detection system (IDS) and aims to demonstrate the various steps and methods involved in a machine learning project.

This project explores various anomaly detection algorithms and employs rules like the 80% and Cattell's rules in Principal Component Analysis (PCA) to identify key components contributing to data variation. It also highlights how results differ when data is separated by normal and abnormal labels before partitioning, versus using a mixed dataset.
